# Github Search

Acesse a aplicação e coloque seu username do github para visualizar seu perfil e repositórios de uma maneira criativa.

![Imagem da Aplicação](./src/assets/images/projeto-github.JPG)

## Tecnologias utilizadas

- ReactJS
- React Router DOM
- Styled-components
- API Github
- MaterialUI Icons
